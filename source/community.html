Community is a large part of being a PHP developer. If you aren't involved in some way, you're missing out on a great resource. We're glad to be apart of the world wide herd and have been able to make so many great friends.

  • PHP.UserGroup - World Map of PHP User Groups
  • PHP Women - An inclusive, global user group providing a support network within the PHP community.
  • User Group Events - Worldwide PHP event calendar.
  • Conferences Call for Papers - Submit to conferences and help teach PHP to others
  • NomadPHP - PHP User Group for those those without local groups. (And to extend those that do!)
